O program  consistem em  fazer  o aterisco correr conforme a entrada  de uma numero, obtendo a base na quantidade e altura..
O programa foi feito em java  na plataform  (IDE) intellij.
Com uma pergunta:
Digite um numero:
Você digite  e vai ter o resulta.
Com 1 for sendo linha.
Com mai dois for de colunas.
Um lendo a quantidade de linha.
o outro colocando a posicçao do asterisco.
e  outro  deslcocando o asterisco para  a direita.
